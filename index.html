Projects

Phone Bill Calculator

An application created with Python using TkInter and Pyperclip library.
A user can dynamically add or remove lines or columns, and after calculating the final bill, they can copy the details to clipboard and paste it somewhere else.
I personally used it, and it allows me to calculate bills around 20% faster.

Instagram Bot

A CLI application created with Python using Selenium API and pandas library.
A user can run the bot, and it will look for posts that has certain hashtags. It will then like the post, add a comment, and follow the poster. It will also record the link of the liked post as well as the comment added to a CSV file.
Click here to visit the Github repo!
